# Data Science Foundations

## Overview
This repository contains hands-on Jupyter notebooks created as part of my Data Science learning journey. The notebooks cover core concepts required for data analysis and machine learning.

## Topics Covered
- NumPy fundamentals for numerical computing
- Pandas for data manipulation and analysis
- Data visualization using Python libraries
- Exploratory Data Analysis (EDA)
- Inferential statistics and hypothesis testing
